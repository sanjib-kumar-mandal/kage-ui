Component

Calendar

The Calendar component lets users view and select dates or ranges in a monthly grid layout. It supports features like week numbers, inline display, custom date formats, time selection, and locale customization for internationalization.

Inputs

List of Input Parameters.

Name

Default Value | Type

selectionMode

'single' | 'range' | 'multiple'. Default 'single'

showTimePicker

Type `boolean`. Default false

timeFormat

'12h' | '24h'. Default '24h'

locale

Local. Default `LOCALE_ID`
