Introduction

Kage-UI is a modern, lightweight, and customizable Angular component library designed with a border-first design philosophy. Built with Angular v19, it offers a set of minimalistic UI components that help you create consistent and visually clean interfaces with ease.

Whether you're building dashboards, admin panels, or product interfaces, Kage-UI aims to deliver a developer-friendly experience with simplicity at its core. All components are fully responsive, accessible, and designed to be easily themeableâ€”making it the perfect choice for scalable Angular applications.

Key Features

  • Clean and minimal component designs
  • Border-based styling for modern aesthetics
  • Fully compatible with Angular v19+
  • SSR-ready and optimized for performance
  • Easy theming and customizability
  • Lightweight and tree-shakable package

Installation

Step 1: Install the Package

npm i kage-ui

Note: @angular/core and @angular/common are required as peer dependencies.

Step 2: Include SCSS File for Customization

To have full control over the UI styles, download the SCSS configuration file: SCSS Config file

Step 3: Use Kage-UI Components

<kage-button color="primary">Click Me</kage-button>
